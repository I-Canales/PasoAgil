## Descripción General
Este repositorio contiene la documentación técnica, el diseño del prototipo y la planificación de aseguramiento de calidad para la plataforma "Paso Ágil". El sistema tiene como objetivo automatizar la captura de datos vehiculares y migratorios en el Complejo Los Libertadores para reducir los tiempos de espera.

## Estructura de Entregables
* `/prototipo`: Enlaces e instrucciones de uso de la interfaz web.
* `/pruebas`: Plan de calidad bajo la norma ISO/IEC 25000 y casos de prueba.
* `/versionado`: Historial de control de cambios justificado por iteraciones.
* `/coevaluacion`: Revisión técnica intergrupal.
